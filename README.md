> Transfertube is a webapp to easily exchange data between device. It is build with angular and uses websockets for server client communication.

### âœ¨ [Demo](https://iqwertz.github.io/Transfertube/)

## Install

```sh
npm install
```

## Usage

```sh
ng serve
```

## Features

- [x] Create Session
- [x] Join Session
- [x] Send Messages
- [x] Format Links
- [x] Send Pictures
- [x] Send Documents
- [x] Tap on message to copy
- [x] Search messages
- [x] Session Menu
- [x] User Alerts
- [x] Fully Responsive
- [ ] Quick paste button
- [ ] Share Session
- [ ] Share Message in Session
- [ ] Session Host
- [ ] Custom uuIds to reconnect as same user
- [ ] Loader when uploading pictures / documents
